# MinecraftConsole-Vulkan

`MinecraftConsole-Vulkan` is a cleaned export branch focused on getting the Vulkan and Apple/macOS work into a standalone Git history that is practical to clone and push.

## Status

This repository is still early and does not represent the full original project history or complete asset set.

## Repository Notes

Some large binary audio and music assets were intentionally omitted from this exported history to keep the repository publishable on GitHub.

## macOS Notes

For macOS-specific setup, build, and run instructions, see [Minecraft.Client/MacOSX/README.md](Minecraft.Client/MacOSX/README.md).
